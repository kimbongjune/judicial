# 11. 화면 설계

## 1. 개요

### 1.1 디자인 철학
- **간결함**: 복잡한 법률 정보를 쉽게 이해할 수 있는 UI
- **효율성**: 필요한 정보에 빠르게 접근
- **접근성**: 법률 전문가와 일반인 모두 사용하기 쉬운 인터페이스

### 1.2 기술 스택
| 구분 | 기술 | 용도 |
|------|------|------|
| 템플릿 엔진 | Jinja2 | 서버사이드 렌더링 |
| CSS | TailwindCSS 3.x | 스타일링 |
| JavaScript | Alpine.js 3.x | 클라이언트 상호작용 |

### 1.3 컬러 팔레트
| 용도 | 색상 | Tailwind |
|------|------|----------|
| 주요 (Primary) | 인디고 | `indigo-600` |
| 보조 (Secondary) | 슬레이트 | `slate-600` |
| 성공/원고승 | 에메랄드 | `emerald-500` |
| 경고/일부승 | 앰버 | `amber-500` |
| 오류/패소 | 로즈 | `rose-500` |
| 배경 | 슬레이트 | `slate-50` |

### 1.4 페이지 구조
```
/                           # 메인 (통합 검색)
├── /search                 # 통합 검색 결과
│   ├── ?tab=cases         # 판례
│   ├── ?tab=constitutional # 헌재결정례
│   └── ?tab=interpretations# 법령해석례
├── /cases/{id}            # 판례 상세
├── /constitutional/{id}   # 헌재결정례 상세
├── /interpretations/{id}  # 법령해석례 상세
└── /similarity            # AI 유사 검색
```

### 4.2 결과 카드 구성
| 요소 | 설명 | 스타일 |
|------|------|--------|
| 아이콘 | 📑 문서 아이콘 | - |
| 사건명 | 클릭 시 상세로 이동 | `text-lg font-semibold` |
| 판결결과 배지 | 원고승(초록), 피고승(로즈), 일부승(앰버) | `rounded-full px-2 py-1` |
| 사건종류 배지 | 민사, 형사, 행정 등 | `bg-slate-100` |
| 메타정보 | 법원 · 사건번호 · 선고일 | `text-sm text-slate-500` |
| 요약 | 판결요지 2줄 | `text-slate-600 line-clamp-2` |
| 인용횟수 | 💬 인용 N회 | `text-indigo-600` |
| 북마크 | ⭐ 클릭 시 토글 | `hover:text-amber-500` |

### 4.3 필터 옵션

**법원:**
- 대법원, 고등법원, 지방법원, 특허법원, 행정법원, 가정법원, 헌법재판소

**사건종류:**
- 민사, 형사, 행정, 가사, 특허

**판결결과:**
- 원고승, 원고일부승, 피고승, 상고기각, 파기환송, 각하, 무죄, 유죄, 기타

**기간:**
- 전체, 최근 1개월, 6개월, 1년, 3년, 5년, 직접입력

### 5.2 탭 구성
| 탭 | 아이콘 | 내용 |
|----|--------|------|
| 요지 | 📌 | 판시사항 + 판결요지 (기본 탭) |
| 전문 | 📄 | 판결문 전체 텍스트 |
| 참조조문 | ⚖️ | 인용된 법령 조문 목록 |
| 참조판례 | 🔗 | 이 판례가 인용한 다른 판례 |
| 인용판례 | 💬 | 이 판례를 인용한 다른 판례 |
| 유사판례 | 🤖 | AI 분석 유사 판례 (유사도 점수) |

### 5.3 우측 사이드바

- **유사판례**: 유사도 점수와 함께 표시

## 11. 반응형 디자인

### 11.1 브레이크포인트
| 브레이크포인트 | 너비 | 레이아웃 변경 |
|---------------|------|--------------|
| sm | 640px | 모바일 메뉴 |
| md | 768px | 2컬럼 그리드 |
| lg | 1024px | 필터 사이드바 표시 |
| xl | 1280px | 3컬럼 그리드 |

### 11.2 모바일 레이아웃
- 헤더: 햄버거 메뉴
- 필터: 하단 슬라이드 패널
- 결과: 단일 컬럼
- 상세: 탭 → 아코디언
